Welcome to the NES Game API!


Through this API, you can:

1. Get a random game.

Sample Query:

{{url}}/api/random

2. Get all the games.

Sample Query:

{{url}}/api/all

3. Search a game by name.

Sample Query:

{{url}}/api/search?title=1942

4. Update game genre.

Sample Query:

{{url}}/api/update-genre/2?new_genre=Platform

5. Add a game.

Sample Query:

{{url}}/api/add

Body need to be provided according to documentation

6. Delete a game.

Sample Query:

{{url}}/api/delete/714?api_key={api_key}
